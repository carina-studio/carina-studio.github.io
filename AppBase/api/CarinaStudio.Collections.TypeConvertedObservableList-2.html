Class TypeConvertedObservableList<TSrc, TDest>

System.Collections.Generic.IList<T> which wraps another System.Collections.Generic.IList<T> and convert elements to another type. Elements will be updated automatically if source System.Collections.Generic.IList<T> implements System.Collections.Specialized.INotifyCollectionChanged interface.
